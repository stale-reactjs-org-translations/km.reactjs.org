### តើខ្ញ៉ំអាចសរសេរ comments នៅក្នុង JSX ដោយរបៀបណា? {#how-can-i-write-comments-in-jsx}

```jsx
<div>
  {/* Comment សរសេរនៅទីនេះ */}
  Hello, {name}!
</div>
```

```jsx
<div>
  {/* អ្នកក៏អាចសរសេរ
    Comment បានច្រើនជួរផងដែរ */}
  Hello, {name}!
</div>
```
